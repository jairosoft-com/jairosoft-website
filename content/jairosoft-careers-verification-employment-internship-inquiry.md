# Jairosoft - Verification of Employment & Internship Inquiry

## Verification of Employment

To request a verification of employment for a current or former Jairosoft employee, please fill up and submit with attached signed and dated authorization from the employee.

Our HR team will process requests in a timely manner. If additional details are required, please specify them in your request. For further assistance, please contact HR at **hr@jairosoft.com**

### Verification of Employment Form

The page includes a comprehensive employment verification form with the following required fields:

#### Required Fields (*)
- **First Name** - Text input with person icon
- **Last Name** - Text input with person icon  
- **Position** - Text input with person icon
- **Specific Request** - Text area for detailed request information
- **Email** - Text input with person icon
- **Authorization File** - File upload button with file icon

#### Form Submission
- Privacy policy agreement checkbox: "I agree to have my information stored Privacy Policy."
- **Submit** button to process the verification request

## Internship Inquiry

### Overview
**Internship Inquiry** - Jairosoft welcomes internship inquiries from students looking to gain hands-on experience in the industry. To apply, please submit your request with the following required documents:

### Required Documents

#### 📄 **Resume**
- Highlight your skills, education, and relevant experiences.

#### 📝 **Cover Letter** 
- Introduce yourself and express your interest in interning Jairosoft Inc.

#### 🏫 **Intent Letter from School**
- A formal endorsement from your institution confirming your internship requirement.

#### 💼 **Portfolio**
- Showcase any relevant projects or work samples.

### Internship Application Form

The internship inquiry form includes the following required fields:

#### Required Fields (*)
- **First Name** - Text input with person icon
- **Last Name** - Text input with person icon
- **Email** - Text input with email icon
- **School** - Text input with school icon
- **Course** - Text input with school icon
- **Render Time Hours** - Time picker field (format: : : - : :)
- **Attached File** - File upload with note: "Kindly provide all documents in a ZIP or RAR file."
- **OJT Coordinator Full Name** - Text input with person icon
- **OJT Coordinator Email** - Text input with email icon

#### Form Submission
- Privacy policy agreement checkbox: "I agree to have my information stored Privacy Policy."
- **Submit** button to process the internship application

## About Jairosoft

Jairosoft is comprised of highly competent Agile Teams with Sitecore certified architects, project managers, full-stack and software test developers that will focus on end-to-end implementation. We are teams of experts in requirements gathering and analysis, written Use Case and Stories, used TFS for managing projects in Agile/SCRUM methodology, and currently implementing SAFe. Our developers are versatile in web development technologies i.e. WebAPI, JSON, ASP.NET Framework, ASP.Net Core, C#, Azure Cloud, XML, HTML, JavaScript, and CSS, Angular JS, and Knockout JS. Well-experienced in Software Design and Architecture including, but not limited to Object-Oriented Analysis and Design, Design Patterns, Domain-Driven Design, SOLID Principles, and N-Tier design. Our software will be tested by our experienced Quality Assurance team including, but not limited to Test Driven Development (TDD), Unit Testing, Selenium IDE, Atlassian JIRA.
